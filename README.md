# Data Jobs Dashboard (Power BI)

Interactive Power BI dashboard analyzing data job postings with a focus on **salary insights, hiring trends, remote share, benefits**, and **geographic distribution**. Includes **drill-through by job title** for deeper analysis.

## What’s inside
- KPI overview (job volume, salary metrics, remote ratio, etc.)
- Salary comparisons & distributions by role
- Posting trends over time
- Job attributes (remote/benefits)
- Location-based views + drill-through page

## Tools
- Power BI Desktop
- Power Query (data cleaning & transformation)

## File
- `Data_Jobs_Dashboard.pbix`

## How to view
1. Open the `.pbix` in Power BI Desktop  
2. Use slicers/filters to explore  
3. Right-click a job title → **Drill through** for role-level details
